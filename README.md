# My_proyect_7
Proyecto modulo 7 Tripelten

En esta oportunidad el proyecto a realizar involucra crear una aplicacion web intercativa, que permita explorar datos de anuncios de venta de vehículos. El propósito principal de este proyecto es poder realizar un análisis visual de los datos mediante gráficos. 

La funcionalidad de esta aplicación web es poder visualizar histogramas y gráficos de dispersion interactivos, lo anterior al accionar botones creados para la generación de estos gráficos. Los graficos creados permiten analizar la relacion entre variables que podemos comparar, al haber revisado inicialmente la estructura de los datos, por ejemplo variables como dias listados vs precio.

Se agregaron, a este proyecto, casillas de verificacion, lo cual permite mantener activa o no una opcion, en este caso algunas de las generaciones de graficos posibles definidas en el aplicativo web.

Para este proyecto se importaron librerias como streamlit, pandas y plotly, las cuales son fundamentales para la creacion de aplicativo web, graficos y para el manejo de datos respectivamente.

Como resultado se obtiene un aplicativo web donde pudimos poner en prática lo aprendido en esta lección.
